
# Stock & Revenue Insights Dashboard

An interactive Python dashboard that visualizes stock prices and quarterly revenue data for Tesla and GameStop. This project fetches real-time stock data using **yFinance** and scrapes revenue data from reliable online sources, then presents insights with **Plotly** interactive charts.

## Features
- Fetches historical stock prices for Tesla (TSLA) and GameStop (GME).
- Scrapes quarterly revenue data from online sources.
- Cleans and formats datasets for analysis.
- Interactive Plotly dashboard showing stock price vs revenue trends.
- One-page visualization for easy comparison.

## Technologies & Libraries
- Python 3.x
- [yFinance](https://pypi.org/project/yfinance/)
- [Requests](https://pypi.org/project/requests/)
- [BeautifulSoup4](https://pypi.org/project/beautifulsoup4/)
- [Pandas](https://pandas.pydata.org/)
- [Plotly](https://plotly.com/python/)
